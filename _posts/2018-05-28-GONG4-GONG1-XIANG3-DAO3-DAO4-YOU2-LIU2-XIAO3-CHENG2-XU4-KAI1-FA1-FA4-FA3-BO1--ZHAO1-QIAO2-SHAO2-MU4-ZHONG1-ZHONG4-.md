项目描述
一、项目描述：
1、基于景区景点的共享导游服务业务，支付及通知与微信公众号相关联；
2、产品定位为以大数据&云计算为游客与导游提供撮合服务的交易工具；
3、后台数据和接口全部基于 discuz 实现，需要开发者懂 PHP 和 discuz api 开发。

二、主要功能点：
1、名片、找导游、找游客、预约、订单等；
2、用户身份验证、微信用户获取管理；


三、可参考产品：
导游宝： http://www.5adao.cn
阳光车导：http://www.jryghq.com

四、人员要求：
1、山西太原的公司或个人，需要到我们公司面谈细节、驻场开发；
2、精通NodeJS或PHP，熟悉jQuery、Javascript、Maven、Redis等技术，熟练使用MySQL等关系型数据库等；
3、良好的沟通能力和契约精神。
