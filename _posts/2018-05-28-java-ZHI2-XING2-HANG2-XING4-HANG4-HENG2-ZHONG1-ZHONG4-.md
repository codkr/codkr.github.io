小型图书馆借书系统后台开发（java）
一、项目描述：
系统主要用于小型书屋（或者咖啡馆）书籍的的日常管理。项目基于java的后台框架开发，为前端提供api数据支持，整体采用前后端分离的架构。
需要工程师白天工作时间可以提供支持，时间充裕倾向于有经验的java后台工程师独立完成。数据库采用的mysql。

二、主要功能点：
查看图书馆信息，最新上架书籍推送，上架图书，发起借书，接受借书，设定借书条件，还书等功能。


三、人员要求：
1、精通java，熟悉流行spring mvc等主流框架，熟练使用MySQL数据库等；
2. 有图书管理系统，企业oa系统开发经验优先
3，良好的沟通能力和契约精神。
4，有充足，稳定时间保证进度。
